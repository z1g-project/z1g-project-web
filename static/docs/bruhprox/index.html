# Welcome Welcome to the BruhProx Documentation! Here you may find information to deploy and run BruhProx yourself! # BruhProx A Quick and Fast proxy used for evading internet censorship or accessing websites in a controlled sandbox using the power of service-workers and more! [**GitHub**](https://github.com/z1g-project/BruhProx) ## Setup BruhProx works by intercepting HTTP requests with a service worker script that follows the TompHTTP specifications. More information regarding TompHTTP can be found [here](https://github.com/tomphttp/specifications). The components of BruhProx are as follows: - BareServer V1 TompHTTP - BruhProx-Static - BruhProx (Production Ready Repo) See the changelog if you're using v6.0 or above! These Instructions **DO NOT APPLY** to BruhProx 5 and older Versions ## Building 1. Clone the repository ```sh $ git clone https://github.com/z1g-project/BruhProx.git > Cloning into BruhProx... $ cd BruhProx ``` 2. Install dependencies ```sh $ npm install ``` 3. Produce the `dist/` directory and compile scripts ```sh $ npm run build ``` ### I don't care about building, where are the scripts for my website?! You can extract the scripts from the NPM package in the [releases section](https://github.com/z1g-project/BruhProx/releases). All the scripts you would typically look for (`uv.bundle.js`, `uv.handler.js`, `uv.sw.js`, and `uv.config.js`) are found within the `dist` directory. If you cannot open the tar.gz (Chrome OS?), use a [converter](https://cloudconvert.com/tar.gz-to-zip). We can only do so much to make the scripts accessible. ## Installation > This will provide built files and the JS API to access the path of dist files. Currently, we don't have an NPM package. As a current solution, you can install from a [GitHub release](https://github.com/z1g-project/bruhprox/releases). Copy the link to the `.tgz` and install using NPM. Soon I plan on adding the package to a zip to save people the hastle but for now follow the instructions ````sh $ npm install https://github.com/z1g-project/BruhProx/releases/download/BruhProx-v6.0/bruhprox-core-v6.tgz ## Packaging This is intended for releasing on hosting the package as a file, not on NPM. ```sh $ npm pack ```` Package will be named `bruhprox-core-X.X.X.tgz` ## Changelogs ### BruhProx 6.1 **Whats New in BruhProx 6.1?** - Fixed Issues with Bare - Fixed Not Secure issue (just use https://[URL] to fix) - Added Cyclic Support - Updated Engine - Made an Official NPM Client - Made BruhProx more Resource Efficient Released: 3/31/2023 ## Deployment
